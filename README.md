# 🥗 Food Expiry Tracker

A smart food inventory management application that helps users track food items, monitor expiry dates, receive timely reminders, scan barcodes, and discover recipes using ingredients before they expire.

---

## 🚀 Features

- ✅ User Authentication (Login & Register)
- 📦 Food Inventory Management
- ⏰ Expiry Date Tracking
- 🔔 Expiry Notifications & Reminders
- 📷 Barcode Scanner for Product Entry
- 🍳 AI Recipe Suggestions
- 📊 Dashboard with Food Status
- 🔍 Search & Filter Food Items
- 📱 Responsive User Interface

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/food-expiry-tracker.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend directory.

```env
PORT=5000

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

GEMINI_API_KEY=your_api_key
```
